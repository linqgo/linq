# Linq for Go

Linq for Go is a data manipulation library inspired by [.Net Linq](dot-net-linq).

Visit the [Functional Catalog](catalog) for a handy categorised function list.

Full documentation may be found at <https://pkg.go.dev/github.com/linqgo/linq>.


[^dot-net-linq]: https://learn.microsoft.com/en-us/dotnet/standard/linq/
[^catalog]: https://github.com/linqgo/linq/blob/main/doc/catalog.md
